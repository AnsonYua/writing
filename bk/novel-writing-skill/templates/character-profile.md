# 角色档案文档模板

本模板定义了单个角色的完整档案格式,用于详细记录小说中所有重要角色的信息。

---

## 文档命名
`character-profiles.md`（所有角色档案合并在一个文档中）

## 文档作用

### 核心价值
- **人物立体化**:确保角色有血有肉,避免脸谱化
- **行为一致性**:角色行为符合其性格和背景
- **关系清晰化**:记录角色间的关系及变化
- **创作参考**:写作时快速查询角色细节

### 适用场景
- 故事大纲完成后创建
- 新增重要角色时补充
- 角色设定修改时更新
- 每次创作前必须读取

## 使用说明

### 何时使用本模板

**强制使用**:
- 故事大纲完成后必须创建
- 执行 `/角色开发` 指令时使用

**可选更新**:
- 新增重要角色时补充
- 角色设定调整时修改

### 填写建议

1. **主角档案必须详尽**:所有项目都应填写,越详细越好

2. **配角档案分级**:
   - 核心配角:填写完整档案
   - 普通配角:简化档案(基本信息+外貌+性格+作用)
   - 工具人:仅记录基本信息和作用

3. **反派要有合理性**:好的反派要有自己的逻辑和动机,不能为了坏而坏

4. **人物关系是动态的**:随着故事发展,角色关系会变化,要及时更新

5. **小传很重要**:用几句话概括角色,有助于把握角色本质

### 与其他文档的关系

- **与novel-setting.md**:设定中的主角信息会在这里详细展开
- **与character-relationships.md**:这里是单个角色的详细信息,关系图是角色间的关系网络
- **与story-outline.md**:角色是为情节服务的,要确保角色行动符合大纲安排

## 角色质量检查清单

完成角色档案后,使用此清单检查:

### 主角检查
- [ ] 基本信息:外貌、年龄、身份已明确
- [ ] 性格:核心性格、优缺点、行为准则已定义
- [ ] 背景故事:家庭、成长经历、关键事件已完整
- [ ] 能力设定:核心能力、能力等级、特殊能力已说明
- [ ] 目标动机:短期目标、长期目标、核心动机已清晰
- [ ] 人物关系:喜欢、讨厌、敬畏、保护的人已列出
- [ ] 成长轨迹:起点、成长节点、终点已规划
- [ ] 人物小传:一句话总结和角色评价已撰写

### 配角检查
- [ ] 基本信息:身份、与主角关系已明确
- [ ] 外貌特征:最显著的特征已描述
- [ ] 性格特征:核心性格已定义
- [ ] 角色作用:在故事中的功能已清晰
- [ ] 人物小传:一句话总结已撰写

### 反派检查
- [ ] 基本信息:身份、实力已定义
- [ ] 反派动机:做坏事的原因已合理
- [ ] 与主角关系:冲突本质已清晰
- [ ] 角色意义:象征意义和对主角的作用已说明

### 一致性检查
- [ ] 所有角色性格前后一致
- [ ] 所有角色能力符合世界观设定
- [ ] 所有角色关系清晰不矛盾
- [ ] 主角与配角的功能不重叠
