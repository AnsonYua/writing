# 题材-风格组合建议参考

本文档提供常见题材与写作风格的匹配建议，供AI在风格建议流程中快速参考。

---

## 题材-风格匹配建议表

### 都市神豪/打脸类

**推荐风格**:市井烟火风 或 人性化语言

**推荐理由**:
1. 都市题材需要接地气，市井风的语言更贴近生活
2. 主角从底层逆袭，市井风能增强代入感
3. 对话自然生动，避免AI味过重
4. 符合番茄读者的阅读习惯

**核心卖点设计**:
- 开篇前800字：主角被羞辱/看不起
- 金手指展示：立即获得系统/财富
- 打脸节奏：每章至少2-3个小爽点
- 对话特点：口语化，有生活气息

**代表作品**:
- 《开局签到亿万资产》
- 《神豪从签到开始》

---

### 苟道流修仙类

**推荐风格**:幽默风格 或 人性化语言

**推荐理由**:
1. 苟道流主角往往低调、谨慎，幽默感能化解紧张
2. 读者喜欢轻松愉快的阅读体验
3. 搞笑的内心独白是苟道流的特色
4. 避免过于严肃的叙述

**核心卖点设计**:
- 开篇前800字：主角在危险环境中谨慎行事
- 苟住的理由：通过内心独白展示
- 反转爽点：关键时刻展现实力
- 对话特点：主角表面恭敬，内心吐槽

**代表作品**:
- 《苟在初圣魔门当人材》
- 《苟在武道世界成圣》

---

### 都市灵异/悬疑类

**推荐风格**:人性化语言（偏严肃）

**推荐理由**:
1. 悬疑题材需要氛围感，避免过度幽默
2. 描写要细腻，增强代入感
3. 对话要符合人物身份和场景
4. 叙述节奏要紧凑但不慌乱

**核心卖点设计**:
- 开篇前800字：直接进入案件/灵异现场
- 悬念设置：每章结尾留钩子
- 案件单元：3-5章一个案件
- 对话特点：职业性强，符合人物身份

**代表作品**:
- 《捞尸人》
- 《我不是戏神》

---

### 传统修仙/玄幻类

**推荐风格**:模仿特定作家（如我吃西红柿）或 人性化语言

**推荐理由**:
1. 传统修仙读者习惯了特定叙述方式
2. 战斗场面需要节奏感
3. 升级体系要清晰明了
4. 世界观介绍要自然融入剧情

**核心卖点设计**:
- 开篇前800字：展示主角天赋或遭遇
- 金手指：外挂要明确（功法/物品/系统）
- 升级节奏：每5-10章一个小突破
- 对话特点：符合修真界设定，半文半白

**代表作品**:
- 《玄鉴仙族》
- 《没钱修什么仙？》（搞笑版）

---

### 重生都市/复仇类

**推荐风格**:人性化语言 或 市井烟火风

**推荐理由**:
1. 都市题材需要生活气息
2. 复仇爽感需要快节奏叙述
3. 情感描写要真实自然
4. 避免过度煽情

**核心卖点设计**:
- 开篇前800字：重生场景或立即复仇
- 先知优势：快速改写命运
- 打脸节奏：高频小爽点+中大爽点
- 对话特点：接地气，有时代感

**代表作品**:
- 《都重生了谁考公务员啊》
- 《重生之都市仙尊》

---

### 女频言情/甜宠类

**推荐风格**:文青清新风 或 人性化语言

**推荐理由**:
1. 情感细腻需要文艺表达
2. 甜宠氛围需要温柔笔触
3. 女性读者偏好细腻描写
4. 避免粗俗语言

**核心卖点设计**:
- 开篇前800字：男女主相遇或冲突
- 情感线：甜点密集，虐点适度
- 对话特点：暧昧感强，细节丰富
- 撒糖：每章至少2-3个甜点

**代表作品**:
- 《林娘子市井生活》
- 《偏执暴君的掌心宠》

---

### 女频重生/复仇类

**推荐风格**:人性化语言（偏犀利）

**推荐理由**:
1. 复仇需要干练的叙述风格
2. 女主成长线要清晰
3. 打脸要干脆利落
4. 情感线与复仇线平衡

**核心卖点设计**:
- 开篇前800字：重生或死亡场景
- 复仇节奏：步步为营，逐渐揭幕
- 人设：智商在线，不白莲
- 对话特点：犀利但不过分

**代表作品**:
- 《重生后我成了权臣的掌中娇》

---

### 科幻/星际类

**推荐风格**:人性化语言（偏冷静）

**推荐理由**:
1. 科幻需要客观冷静的叙述
2. 技术设定要清晰明了
3. 避免过度情绪化
4. 节奏要快但有序

**核心卖点设计**:
- 开篇前800字：展示世界观或冲突
- 科幻设定：自然融入剧情
- 战斗/冒险：紧张刺激
- 对话特点：符合科幻背景

**代表作品**:
- 相关星际、机甲类作品

## 平台-风格匹配原则

### 起点中文网
- **风格要求**:可接受复杂设定，文笔要求相对高
- **推荐风格**:人性化语言、模仿特定作家
- **避免**:过度市井化

### 番茄小说
- **风格要求**:接地气，快节奏，爽点密集
- **推荐风格**:市井烟火风、幽默风格、人性化语言
- **避免**:过度文艺、慢节奏铺陈

### 晋江文学城
- **风格要求**:情感细腻，对话精彩
- **推荐风格**:文青清新风、人性化语言
- **避免**:粗俗语言、过度简洁

## 快速决策流程

当AI需要快速给出风格建议时，遵循以下逻辑：

```
IF 题材 = 都市神豪/打脸:
    → 推荐:市井烟火风
ELSE IF 题材 = 苟道流修仙:
    → 推荐:幽默风格
ELSE IF 题材 = 都市灵异/悬疑:
    → 推荐:人性化语言(偏严肃)
ELSE IF 题材 = 传统修仙/玄幻:
    → 推荐:人性化语言
ELSE IF 题材 = 女频言情:
    → 推荐:文青清新风
ELSE IF 题材 = 女频重生:
    → 推荐:人性化语言(偏犀利)
ELSE:
    → 推荐:人性化语言(通用)
END IF

IF 平台 = 番茄:
    → 倾向市井/幽默风格
ELSE IF 平台 = 起点:
    → 倾向人性化/模仿作家风格
ELSE IF 平台 = 晋江:
    → 倾向文青/人性化风格
END IF
```

## 特殊情况处理

### 用户上传样本
- 优先分析样本风格
- 基于样本给出改进建议
- 说明样本风格与推荐题材的匹配度

### 用户指定风格
- 尊重用户选择
- 说明该风格的优缺点
- 给出具体应用建议

### 用户要求模仿特定作家
- 分析该作家风格特点
- 结合用户题材给出应用建议
- 提醒：借鉴风格不抄袭情节
