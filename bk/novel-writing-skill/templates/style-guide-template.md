# 风格指南生成模板

本模板定义了如何基于用户选择自动生成 style-guide.md 文件。

---

## style-guide.md 标准格式

### 文件命名
`style-guide.md`

### 生成时机
- 用户完成3轮询问引导并最终确认后
- 用户直接选择题材和风格后

## 基于选择的风格映射

### 人性化语言
**核心特征**:
- 句式特点：避免过于规整，增加长短变化
- 用词习惯：口语化表达，减少公式词
- 叙述节奏：自然流畅，适当跳跃
- 对话风格：简洁高效，有信息量

**禁止事项**:
- 避免"首先、其次、最后"等公式词
- 避免每段长度相似
- 避免过度完美的逻辑性

### 幽默风格
**核心特征**:
- 句式特点：短句为主，节奏明快
- 用词习惯：使用搞笑比喻和夸张
- 叙述节奏：张弛有度，穿插轻松时刻
- 对话风格：机智抖机灵，幽默风趣

**禁止事项**:
- 严肃场景不能随便插科打诨
- 幽默要适度，不要变成搞笑文
- 要符合人物性格

### 市井烟火风
**核心特征**:
- 句式特点：口语化强，接地气
- 用词习惯：生活化细节，方言俗语适度使用
- 叙述节奏：平民视角，不高高在上
- 对话风格：生活化，真实自然

**禁止事项**:
- 避免装腔作势的文艺表达
- 避免过度使用方言（读者看不懂）
- 避免脱离生活实际

### 文青清新风
**核心特征**:
- 句式特点：细腻感性，意境优美
- 用词习惯：文字有诗意，情感描写深入
- 叙述节奏：慢节奏铺陈，注重氛围
- 对话风格：含蓄委婉，情感细腻

**禁止事项**:
- 不要过度抒情，影响节奏
- 网文读者更喜欢情节，不要本末倒置
- 适度使用，点到为止

## 生成逻辑

AI需要根据用户选择填充模板：

```python
def generate_style_guide(platform, genre, satisfaction, style):
    # 1. 确定风格特征
    style_features = get_style_features(style)

    # 2. 基于题材调整建议
    genre_specific = get_genre_specific_advice(genre, style)

    # 3. 基于平台调整建议
    platform_specific = get_platform_specific_advice(platform, style)

    # 4. 生成完整指南
    guide = fill_template(style_features, genre_specific, platform_specific)

    return guide
```

---

## 注意事项

1. **个性化**: 每个用户的 style-guide.md 都是不同的，基于其具体选择生成
2. **具体化**: 提供具体的示例和对比，不要泛泛而谈
3. **可执行**: 给出的建议必须是AI可以实际应用的
4. **约束性强**: 这是强约束文档，后续创作必须严格遵守
