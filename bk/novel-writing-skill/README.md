# 网文与爆款短剧创作 Skill

这个 skill 现在有两条主线：

1. 原有的长篇网文创作流程
2. 新增的“少量关键词直出爆款短剧”流程

如果你只给几个关键词，希望它直接写出一部成熟向、高张力、可持续分集的短剧，它现在应该优先走短剧模式。

## 新增能力

### 1. 关键词直出短剧
输入 3 到 8 个关键词，它会自动补齐：
- 题材
- 关系
- 商业套路
- 情绪方向
- 分集节奏

默认直接产出：
- `short-drama-bible.md`
- `episode-grid.md`
- `title-hook-package.md`
- `scene-001-outline.md`
- `episode-001.md`
- `episode-002.md`
- `episode-003.md`

### 2. 爆款张力构建师
把一个 premise 拆成 3 分钟以内的高张力场景，按：
- Hook
- Build-Up
- Escalation
- Peak
- Cliffhanger

### 3. 爆款热点套路注入
把模糊题材自动叠加高收益套路，优先生成：
- 冷面总裁
- 办公室上下级
- 秘密情人
- 逆袭打脸
- 复仇中动心

### 4. 爆款标题与钩子设计
会根据剧情摘要直接出：
- 多种标题
- taglines
- 3到5秒停留开场

## 推荐使用方式

### 用关键词直接生成
示例：
- `霸总 秘密情人 复仇`
- `办公室 上司下属 禁忌感`
- `豪门 假结婚 高虐 追妻火葬场`

### 想先看概念
让它先做：
- `/套路注入`
- `/标题钩子`

### 想直接出可拍内容
让它直接走：
- `/短剧模式`
- `/关键词生成`

## 风格说明

这里的“成人向”默认不是露骨色情，而是：
- 成熟关系
- 高压拉扯
- 禁忌氛围
- 欲望和报复纠缠
- 让人停不下来的 cliffhanger

核心目标是：少问、快出、够抓马、够好拍、够能追下一集。
