# 短剧快写路径

当用户不是来做完整长篇规划，而是希望“给几个词直接出短剧”时，使用这条路径。

## 适用触发
- 给几个关键词写短剧
- 直接写一部爆款短剧
- 霸总 / 豪门 / 禁忌 / 秘密情人 / 办公室 / 复仇 / 打脸
- 想先拿到可拍版本，而不是慢慢讨论设定

## 输入最低要求
最低只需要 3 个信息：
- 一个题材词
- 一个关系词
- 一个情绪词

示例：
- `霸总 秘密情人 复仇`
- `办公室 上司下属 禁忌感`
- `豪门 前任重逢 高虐`

## 自动补全规则

如果用户只给模糊关键词，自动补全：

### 世界默认
- 现代都市
- 豪门/集团/律所/娱乐公司/酒店业优先

### 关系默认
- 冷面强势男主 + 有韧性的女主
- 表面压制，实际双向失控

### 商业默认
- 24 集
- 每集 2 到 3 分钟
- 每集一个钩子
- 第 1 集必须有危险吸引

## 默认交付包

除非用户只要单一内容，否则第一次交付直接给：
1. 1 个最强概念
2. 3 到 5 个标题/开场钩子方案
3. 短剧圣经
4. 分集表
5. 第一场 3 分钟张力拆解
6. 前 3 集成品脚本

## 成人向处理原则

这里的“成人向”默认理解为：
- 成熟关系
- 禁忌氛围
- 欲望与控制
- 危险亲密
- 强情绪拉扯

不要默认写成露骨色情。优先使用：
- 贴近又克制的动作
- 带刺的对话
- 未说破的占有欲
- 身份地位带来的压迫感
- 即将越界但尚未彻底越界的瞬间

## 高转化核心检查

生成完成后自检：
- 剧名是否一眼能看懂卖点
- 第一场是否 5 秒内有冲突
- 前三集是否连续升级
- 关系张力是否强于世界观说明
- 每集结尾是否能自然接下一集
