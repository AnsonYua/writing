# 短剧脚本格式

## 目标
把剧情写成能直接进入拍摄沟通或继续拆镜的格式。

## 单集标准
- 时长：2到3分钟
- 场景数：2到4场
- 节奏：快
- 每集核心：一个主要冲突、一次关系升级、一个强钩子

## 推荐结构

### 开头（0-20秒）
- 直接冲突或视觉钩子
- 不做背景铺垫

### 中段（20-100秒）
- 核心拉扯
- 信息揭露
- 距离升级

### 结尾（100秒后）
- 反转、失控、身份暴露、危险宣告

## 成稿格式

```markdown
# 第01集《标题》

**时长**：约2-3分钟
**核心情绪**：危险吸引 / 冲突升级 / 禁忌拉扯
**本集钩子**：[一句话]

## 场1【夜 / 内 / 地点】
[环境只写必要信息]

[动作]
角色A：台词
角色B：台词

## 场2【夜 / 内 / 地点】
...

## 结尾钩子
[一句话说明]
```

## 台词规则
- 每句台词必须有功能
- 少解释，多试探、压迫、反刺
- 让不同角色有不同语言习惯
- 尽量让台词短，利于演员打点

## 成熟向关系戏规则
- 用靠近、停顿、触碰、退让、反制表达张力
- 不用堆砌露骨生理描写
- 对白里要有占有、控制、心软、报复的双关

## 每集结尾优先钩子
- “你以为这就结束了？”
- 第三人撞见
- 身份当场反转
- 男主给出更危险的条件
- 女主发现自己被算计
