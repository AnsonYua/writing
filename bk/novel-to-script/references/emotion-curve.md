# 情绪曲线管理

## 核心理念

短剧的本质是「情绪过山车」。观众不是在看故事，是在体验情绪。

情绪曲线设计的铁律：**压力积累 → 爽点释放 → 短暂平息 → 更大压力 → 更强爽点**，螺旋上升，直到大结局。

平坦的情绪曲线 = 观众流失。每一集都很爽 = 审美疲劳，等于每一集都不爽。

---

## 一、全剧情绪曲线模板

### 标准爽剧曲线（60集参考）

```
情绪强度
  ▲
10│                                    🔥🔥🔥终极高潮
  │                              🔥高潮
  │                        🔥
  │              🔥              ↘至暗
  │        🔥         ↘低谷
  │   🔥        ↘低谷
  │🔥
  │ ↘平息  ↘平息
  └────────────────────────────────▶ 集数
   1   10  20  30  40  50  60
       💰付费卡点
```

### 四阶段强度分配

| 阶段 | 集数 | 强度基线 | 节奏特征 | 核心任务 |
|------|------|---------|---------|---------|
| 引爆期 | 1-15% | 中高 | 密集冲突 | 建立人设、初始爽点、吸引观众 |
| 蓄力期 | 15-40% | 中 → 高 | 逐步升温 | 扩大冲突、引入新势力、蓄力 |
| 爆发期 | 40-80% | 高 → 极高 | 高密度反转 | 多线汇聚、连续爽点、大逆袭 |
| 收束期 | 80-100% | 极高 → 收束 | 先快后稳 | 终极对决、情感升华、大结局 |

## 二、关键节点设计

### 必须设计的关键节点

| 节点 | 位置（占总集数比） | 内容要求 |
|------|----------------|---------|
| 爆炸开场 | 第1集 | 30秒内引爆核心矛盾，奠定基调 |
| 初始爽点 | 约5%处 | 主角第一次小胜利，留住观众 |
| 💰付费卡点 | 约15-20%处 | 最强悬念，身份/能力即将揭晓但被截断 |
| 第一转折 | 约25%处 | 主角处境发生根本性变化 |
| 新势力登场 | 约35%处 | 引入更强对手或更复杂局面 |
| 中点大反转 | 约50%处 | 故事走向重大转折，观众预期被打破 |
| 至暗时刻 | 约65%处 | 主角遭遇最大危机，一切似乎失去 |
| 终极反击启动 | 约75%处 | 绝地反击，进入终局模式 |
| 高潮连爆 | 约85-90%处 | 多条线索汇聚，连续爆发 |
| 大结局 | 最后3-5集 | 终极对决 + 情感收束 + 圆满/反转结局 |

### 付费卡点设计要点（最关键）

付费卡点决定平台收益，必须精心设计：

**结构要求：**
- 前几集密集铺垫，反派嚣张到极致
- 主角的真实身份/能力蓄力到临界点
- 最后一刻戛然而止——观众被吊在最高点

**卡点结尾钩子示例：**
```
「就在所有人嘲笑他的时候，会议室的门被推开了——
走进来的，是整个商界都认识的那个人。」
```

## 三、单集情绪设计

### 每集情绪类型标注

在分集目录中，为每集标注情绪类型：

| 情绪类型 | 标记 | 特征 | 使用频率 |
|---------|------|------|---------|
| 爽点爆发 | 💥 | 强反转、打脸、逆袭 | 主力，约40%的集 |
| 压力蓄积 | 🔨 | 主角被压制、反派嚣张 | 约25%，为爽点铺垫 |
| 情感波动 | 💔 | 误解、心动、决裂 | 约20%，调节节奏 |
| 悬念引爆 | ❓ | 真相将近、线索汇聚 | 约10%，在关键节点 |
| 平缓过渡 | ➡️ | 信息交代、关系推进 | 约5%，尽量压缩 |

**核心原则：连续3集不能出现相同情绪类型标记。平缓过渡集（➡️）绝不能连续出现。**

### 单集内部情绪波形

每集2-3分钟内，情绪必须有起伏，不能是一条平线：

```
好的单集波形：
情绪 ▲
     │  ↗高潮
     │↗      ↘
     │          ↗钩子
     └──────────▶ 时间

坏的单集波形（平铺）：
情绪 ▲
     │───────────
     └──────────▶ 时间
```

## 四、情绪曲线追踪文件

分集目录生成时，在目录末尾附加情绪曲线规划表：

```markdown
## 情绪曲线规划

| 集数 | 情绪类型 | 强度(1-10) | 核心情绪事件 | 与前集关系 |
|------|---------|-----------|------------|---------|
| 第1集 | 💥爽点爆发 | 7 | 主角初次反击 | 开篇 |
| 第2集 | 🔨压力蓄积 | 4 | 反派反制，主角陷入被动 | 平息后蓄力 |
| 第3集 | 💥爽点爆发 | 8 | 身份首次半揭露 | 爆发 |
| 第4集 | 🔨压力蓄积 | 3 | 新反派势力登场 | 新一轮蓄力 |
| 第5集 | ❓悬念引爆 | 6 | 关键秘密浮出水面 | 铺垫 |
```

**情绪强度警戒规则：**
- 连续2集强度 ≤ 4：提示「节奏过缓，需检查」
- 连续2集强度 = 10：提示「情绪透支，建议在中间插入平息集」
- 付费卡点集强度必须 ≥ 9

---

## 五、情绪曲线审查时机

### 目录生成时（规划阶段）
- 生成目录后，自动附上情绪曲线规划表
- 检查关键节点是否到位
- 检查是否有连续平缓段（连续3集强度 ≤ 5）

### 每5集完成时（执行阶段）
- 对照情绪曲线规划表，评估实际执行是否偏离
- 如有偏离，说明原因并询问是否调整后续集的情绪设计

### 常见情绪曲线问题

| 问题 | 症状 | 解决方案 |
|------|------|---------|
| 头重脚轻 | 前几集太爽，后期无法超越 | 前期爽点降级，留余量给后期 |
| 中段塌陷 | 15-40集节奏拖沓 | 删减铺垫集，合并场景 |
| 高潮提前 | 最强爽点出现在60%处 | 将最强爽点拆分为两个，分散到75%和90%处 |
| 结局平淡 | 大结局的情绪强度低于中段 | 最后3集必须是全剧情绪最高点 |
