# 专业剧本格式规范（短剧 / 动态漫双标准）

## 两种模式对比

| 维度 | 短剧模式 | 动态漫模式 |
|------|---------|----------|
| 每集字数 | 800-1200字 | 1500-2500字 |
| 每集场次 | 3-5个 | 4-7个 |
| 台词单句上限 | 30字 | 40字 |
| 景别标注 | 关键位置（全集5-8处） | 较详细（全集8-12处） |
| 内心描写 | 禁止，改为台词或动作 | 允许旁白形式，不超过全集20% |
| 开场引爆时限 | 前3行内 | 前5行内 |
| 节奏取向 | 极致压缩，跳切叙事 | 节奏稍慢，对白更完整 |
| 适用平台 | 抖音、快手、小程序 | B站、漫画平台 |

## 标准格式模板（两种模式通用）

```
剧名：《剧名》
集数：第X集  标题
模式：[短剧/动态漫]

════════════════════════════════════════

场次 X-1：内/外  具体地点  日/夜
────────────────────────────────────────
出场人物：角色1；角色2

△ （全景）场景环境描写。交代时间、地点、氛围。

角色1
（语气或动作描写）
台词内容。

△ （近景）角色2的反应动作。

角色2
（语气或动作描写）
台词内容。

△ （特写）关键表情或物件的特写。

角色1
（语气或动作描写）
台词内容。

────────────────────────────────────────

场次 X-2：内/外  具体地点  日/夜
────────────────────────────────────────
出场人物：角色N

△ （全景）新场景。

......

════════════════════════════════════════
本集完 ｜ 下集预告：一句话悬念钩子
```

## 格式要素详解

### 场景头
```
场次 [集数]-[场次序号]：[内/外]  [具体地点]  [日/夜]
```
- 内/外：室内景 / 室外景
- 时间：日 / 夜 / 黄昏 / 清晨
- 示例：`场次 3-2：外  医院门口  夜`

### 出场人物
每个场次紧跟场景头，用分号分隔：
```
出场人物：陈锋；林小雨；林母
```
群演标注：`路人甲；围观群众`

### △ 动作/场景描写

**短剧模式**：每段 △ 不超过3行，纯动态描写，无内心活动

**动态漫模式**：每段 △ 不超过4行，可有简短心理描写（旁白形式）

景别参考：

| 景别 | 适用场景 |
|------|---------|
| （全景） | 开场建立环境、群体场面 |
| （中景） | 两人对话、常规交互 |
| （近景） | 情感表达、重要对话 |
| （特写） | 情绪爆发、关键物件、面部表情 |
| （远景） | 气势场面、环境渲染 |

### 角色台词（三行结构）
```
角色名
（语气/动作指示）
台词正文。
```

**短剧模式**：单句 ≤ 30字，口语化，每句推动剧情

**动态漫模式**：单句 ≤ 40字，可稍有情感铺陈

语气指示示例：（冷笑）（猛地拍桌）（声音颤抖）（低头，语气轻蔑）

### 特殊标记

| 标记 | 含义 | 示例 |
|------|------|------|
| △ | 动作/场景描写 | △ 主角猛地站起身 |
| （） | 语气/动作指示 | （咬牙切齿） |
| 【】 | 画外音/旁白 | 【旁白】时间回到三年前…… |
| ♪ | 背景音乐提示 | ♪ 紧张悬疑配乐渐起 |
| ☎ | 电话/视频通话 | ☎ 手机铃声响起 |

## 每集结构规范

### 短剧模式结构

| 位置 | 占比 | 内容要求 |
|------|------|---------|
| 开场 | 前30秒 | 前3行引爆冲突，承接上集钩子 |
| 发展 | 中间60% | 冲突升级，至少1次反转 |
| 高潮 | 30-45秒 | 爽点释放或情绪顶峰 |
| 钩子 | 最后10秒 | 强悬念，必须让人想看下集 |

### 动态漫模式结构

| 位置 | 占比 | 内容要求 |
|------|------|---------|
| 开场 | 前45秒 | 前5行建立冲突，稍有铺垫空间 |
| 发展 | 中间55% | 冲突发展，可有情感对白 |
| 高潮 | 45-60秒 | 爽点释放，情绪更完整 |
| 钩子 | 最后15秒 | 悬念或情感余韵 |

## 完整短剧示例

```
剧名：《龙王归来》
集数：第1集  三年之约
模式：短剧

════════════════════════════════════════

场次 1-1：内  破旧出租屋  夜
────────────────────────────────────────
出场人物：陈锋；林小雨

△ （全景）逼仄的出租屋，墙皮脱落，灯光昏黄。
  一张折叠桌上摆着两盘简单的素菜和一个小蛋糕。
  蛋糕上歪歪扭扭写着"结婚三周年快乐"。

△ （近景）陈锋坐在桌边，穿着洗得发白的T恤，
  双手粗糙，指节上有旧伤痕。他看着蛋糕，嘴角
  微微上扬。

陈锋
（温柔地）
小雨，这三年，委屈你了。

△ （中景）林小雨从厨房端出一碗汤，围裙上沾着
  油渍，脸上却带着满足的笑。

林小雨
（笑着摇头）
跟你在一起，不委屈。就是……

△ 林小雨的笑容突然僵住。她低头看了一眼手机。

林小雨
（语气骤变，慌张）
我妈又打电话来了……明天的家宴，她让你
一定要到。

△ （特写）陈锋的眼神瞬间变得深邃而复杂，握着
  筷子的手微微收紧。

陈锋
（平静，但目光如刀）
去。该见的人，总要见。

────────────────────────────────────────

场次 1-2：内  林家豪宅·客厅  日
────────────────────────────────────────
出场人物：陈锋；林小雨；林母；林浩天；赵倩

△ （全景）金碧辉煌的客厅，水晶吊灯、真皮沙发、
  名画。一群衣着光鲜的人围坐在一起。
  陈锋穿着那件洗白的T恤，站在门口，格格不入。

♪ 轻蔑、压抑的背景音乐

林母
（扫了一眼陈锋，嫌弃地皱眉）
小雨，你就带了这么个东西回来？

△ 全场哄笑。林小雨攥紧了陈锋的衣角。

林浩天
（翘着二郎腿，弹了弹雪茄灰）
姐，我早说了，这种废物，趁早离了吧。
我给你介绍赵家公子，那才配得上咱林家。

△ （特写）陈锋的表情没有任何变化。只是左手
  无名指上，一枚古朴的黑色戒指微微发出暗光。

☎ 突然，陈锋口袋里的手机震动。

△ 陈锋低头看了一眼——屏幕上只有四个字：
  "龙主，时候到了。"

△ （近景）陈锋缓缓抬头，嘴角勾起一丝冷笑。

陈锋
（声音很轻，但在场所有人都听得一清二楚）
三年期满。该还的，一分不少。

════════════════════════════════════════
本集完 ｜ 下集预告：陈锋的真实身份即将揭晓——
当林家人知道他们羞辱的人是谁，一切都将天翻地覆。
```

## 常用爽点台词套路

| 场景 | 台词模式 |
|------|---------|
| 身份揭露 | 「你知道他是谁吗？」→ 全场震惊 |
| 财力碾压 | 「这栋楼，我买了。」 |
| 关系碾压 | 「把你们总裁叫来。」→「我就是总裁。」 |
| 逆袭宣言 | 「从今天起，换我来。」 |
| 即时打脸 | 反派话音刚落 → 打脸证据出现 |
| 冷静碾压 | （不看对方）「你在跟我说话吗？」 |

## 信息呈现方式：小说视角转剧本视角

小说可以直接写「他心想……」「她感觉……」，剧本没有叙述者，所有信息必须通过**画面、声音、台词**呈现。这是转换最容易出错的地方。

### 四种信息类型的呈现规则

| 小说里的信息类型 | 错误做法 | 正确做法 |
|--------------|---------|---------|
| **心理活动** | 直接改成台词说出来 | 改为动作 + 极短台词，或不呈现 |
| **背景信息**（世界观、历史） | 大段旁白解说 | 融入台词（角色之间的对话带出），或字幕一句话交代 |
| **时间跳跃** | 台词解释「三天后……」 | △ 字幕标注，或用场景变化暗示 |
| **第一人称叙述** | 保留成旁白 | 拆解为具体动作和台词，旁白只在必要时用 |

### 心理活动的三种转化方式

**方式一：动作化（首选）**

```
原著：陈锋看着那份合同，心中百感交集，既有愤怒又有悲凉。

❌ 旁白版：【旁白】看着这份合同，陈锋心中百感交集。
✅ 动作版：△ （特写）陈锋盯着合同，手指缓缓收紧，将纸角压出折痕。
```

**方式二：台词化（心理有明确指向时）**

```
原著：她不想让他看出自己在乎，决定装作若无其事。

❌ 旁白版：【旁白】她决定装作若无其事。
✅ 台词版：
林小雨
（低头整理衣角，语气刻意平淡）
随便。你决定就好。
```

**方式三：直接删除（心理是过渡性的）**

```
原著：陈锋想了想，决定还是去找她谈谈。

→ 直接切到他敲门的画面，不需要交代他「想了想」的过程。
```

### 背景信息的呈现方式

**少量信息：融入对话**
```
❌ 旁白：【旁白】陈家三年前因为一场商业阴谋而破产。
✅ 台词：
林浩天
（轻蔑）
陈家？那个三年前就破产的陈家？
```

**必须交代的时间地点：字幕**
```
△ 字幕：三年后 / 上海
△ 字幕：同一天，凌晨两点
```

**世界观设定（玄幻类）：用角色反应带出**
```
❌ 旁白：【旁白】在这个世界，筑基期是修炼的第一个门槛，极难突破。
✅ 反应版：
长老
（震惊起身）
他……他直接跨越了筑基期？！这怎么可能！
```

## 场景重复处理规则

短剧里，固定场景（主角的家、办公室、某个常用地点）会反复出现。

### 规则：首次出现 vs 再次出现

**首次出现**：完整写场景头 + △ 环境描写，建立画面感。

```
场次 1-1：内  陈锋的简陋出租屋  夜
────────────────────────────────────────
出场人物：陈锋

△ （全景）逼仄的出租屋，墙皮脱落，灯光昏黄。
  折叠桌上摆着两个素菜，角落堆着搬家纸箱。
```

**再次出现**：只写场景头，**直接进台词或关键动作**，不重复环境描写。

```
场次 5-3：内  陈锋的出租屋  夜
────────────────────────────────────────
出场人物：陈锋；林小雨

△ 林小雨站在门口，看着屋内，沉默片刻。

林小雨
（轻声）
你一直住这里？
```

### 场景变化时才重新描写

如果场景发生了明显变化（同一地点，但时间推移后变化了），才需要重新写 △ 环境描写，并突出变化：

```
场次 12-1：内  陈锋的豪华办公室  日
────────────────────────────────────────
出场人物：陈锋

△ （全景）宽阔的落地窗，整个城市尽收眼底。
  这里，就是三个月前那间出租屋的位置。
```

### 熟悉场景的快速标注法

对剧本里频繁出现的场景，可以在首次出现时用括号注明「常驻场景」，后续出现时无需描写：

```
场次 1-1：内  林氏集团总裁办公室（常驻场景）  日
△ （全景）…… [首次完整描写]

场次 4-2：内  林氏集团总裁办公室  日   ← 后续直接省略环境描写
```
