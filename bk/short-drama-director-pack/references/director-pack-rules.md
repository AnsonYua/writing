# Director Pack Rules

Use this file when assembling the last production handoff.

## Prefer These

- one page mindset
- only the details the next stage must know
- hook-first prioritization
- continuity warnings that prevent expensive mistakes

## Avoid These

- repeating the entire script
- repeating every shot if storyboard already exists
- vague praise
- abstract directing language with no concrete use

## Good Character Execution Notes

- keep the uncle's danger in normality, not villain face
- keep the mother's pressure in low voice and hesitation
- keep the girl's alertness visible in hands, eyes, and pauses

## Good Scene Execution Notes

- phone screen must read clearly enough to trigger suspicion
- bedroom-door exchange depends on inside/outside contrast
- final realization should land on the girl's face, not explanation
