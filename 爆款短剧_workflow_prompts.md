# 爆款短剧 Workflow Prompts

這份給你的不是單一 prompt，而是兩套可直接複製用的方案：

- A. 超短一鍵版：一句話直接生成爆款短劇
- B. 六段工作流版：更穩、更容易持續出片

---

## A. 超短一鍵版

適合你想快速測題材、快速跑 10 個 idea。

```text
你是短視頻平台的爆款短劇總編劇，不是寫普通故事，而是要把我的一句話想法，升級成高完播、高情緒、高反轉的 AI 短劇方案。

請按以下規則創作：
- 開頭 3 秒必須有鉤子
- 劇情節奏極快，每一小段都要推進衝突
- 至少包含 2 次強反轉
- 主角要有強烈欲望、弱點、隱藏優勢
- 反派要有壓迫感
- 對白要短、狠、可剪輯
- 整體要適合 AI 漫劇 / AI 短劇出圖與出視頻
- 如果我的點子太普通，你要主動升級成更有爆款感的版本

請輸出：
1. 爆款定位
2. 一句話高概念
3. 3 個核心角色
4. 8 集分集梗概
5. 第一集完整腳本
6. 8 個分鏡
7. 3 條 AI 出圖提示詞
8. 3 條 AI 視頻提示詞

我的一句話創意是：
{{一句話}}
```

## B. 六段工作流版

這套比較接近你貼文裡那些人真正用的方法。

順序是：

1. 爆款樣本分析
2. 題材升級
3. 系列總綱
4. 第一集腳本
5. 分鏡拆解
6. 出圖 / 出視頻提示

不要一開始就直接讓模型又寫故事又寫分鏡又寫視頻，容易亂。分段最穩。

---

## 1. 爆款樣本分析 Prompt

用途：先讓模型學會什麼叫“短劇平台感”。

```text
你是爆款短劇分析師。请分析下面这些短剧标题 / 简介 / 桥段，提炼出它们共同的爆款结构，而不是复述剧情。

你的任务：
- 总结这些样本的共同题材
- 总结最常见的人设组合
- 总结最常见的冲突类型
- 总结最常见的开场钩子
- 总结最常见的反转模式
- 总结最常见的结尾留悬念方式
- 提炼成一个“可复用的爆款短剧公式”

输出格式：
1. 样本共性
2. 爆款钩子公式
3. 人设公式
4. 冲突升级公式
5. 反转公式
6. 结尾钩子公式
7. 一个可直接复用的模板

短剧样本如下：
{{貼上你抓到的標題/簡介/橋段}}
```

---

## 2. 題材升級 Prompt

用途：你只給一句話，模型負責把它升級成更像會火的短劇核心概念。

```text
你是爆款短劇策劃師。請把我的一句話創意升級成更有傳播力、更有情緒張力、更適合短視頻平台的短劇題材。

要求：
- 保留我的核心想法
- 強化身份反差
- 強化衝突
- 強化羞辱、秘密、誤會、背叛、反殺、逆襲中的至少兩項
- 提供 3 個不同方向的升級版本
- 每個版本都要說明它為什麼更容易火

輸出格式：
1. 原始創意的問題
2. 升級方向 A
3. 升級方向 B
4. 升級方向 C
5. 你最推薦的一版
6. 最終一句話高概念

我的一句話創意：
{{一句話}}
```

---

## 3. 系列總綱 Prompt

用途：把一句話高概念變成 8 集到 12 集的連載骨架。

```text
你是一名專做爆款短劇的系列編劇。請根據下面的一句話高概念，設計一個 8 集短劇系列，每集 45-90 秒。

硬性要求：
- 第 1 集前 3 秒就要出鉤子
- 每一集都要有一個新的衝突推進
- 至少每 2 集出現一次反轉
- 每集結尾都要逼觀眾想點下一集
- 整體節奏像短視頻平台，不像傳統電視劇

請輸出：
1. 題材定位
2. 核心爽點
3. 主角/反派/關鍵配角
4. 三幕式總綱
5. 8 集分集設計

每集需包含：
- 集標題
- 開場鉤子
- 本集衝突
- 本集反轉
- 結尾懸念

一句話高概念如下：
{{高概念}}
```

---

## 4. 第一集腳本 Prompt

用途：把分集梗概變成真的能拍、能剪的腳本。

```text
你是短劇腳本編劇。請根據下面的劇情設定，寫出第一集完整腳本。

要求：
- 全集控制在 45-90 秒感受內
- 場景不要太多，2-4 個場景即可
- 對白短、狠、情緒明確
- 每個場景都要推進衝突
- 結尾必須停在最強懸念處
- 適合後續拆分鏡和 AI 視頻生成

輸出格式：
1. 第一集目標
2. 場景列表
3. 完整腳本

完整腳本按以下格式：
- 場景編號
- 地點
- 出場人物
- 畫面動作
- 對白
- 情緒變化
- 本場鉤子

故事設定如下：
{{貼上總綱和第一集梗概}}
```

## 5. 分鏡拆解 Prompt

用途：把腳本變成能出圖、能出視頻的鏡頭語言。

```text
你現在是短劇導演兼分鏡師。請把下面的第一集腳本，拆成 8-12 個適合 AI 漫劇 / AI 短劇 的鏡頭。

要求：
- 以 9:16 竖屏短剧为前提
- 鏡頭切換要快
- 鏡頭目的要明確：鋪墊、壓迫、揭露、反轉、情緒爆發
- 角色外觀保持穩定
- 場景盡量集中，方便統一生成

每個鏡頭輸出：
- 鏡頭編號
- 景別
- 構圖
- 畫面內容
- 人物動作
- 對白/旁白
- 鏡頭目的
- 建議時長

腳本如下：
{{貼上第一集腳本}}
```

---

## 6. 出圖 / 出視頻 Prompt

用途：最後把角色、場景、鏡頭轉成模型可用提示詞。

```text
你是 AI 短劇視覺導演。請根據以下角色設定、場景設定和分鏡，生成統一風格的 AI 出圖提示詞與 AI 視頻提示詞。

要求：
- 角色外觀固定，不要每鏡都變
- 統一風格，例如：電影感、日漫感、寫實漫改、都市豪門、校園青春、懸疑冷調
- 適合 9:16 竖屏
- 明確描述人物、服裝、場景、光線、情緒、動作、鏡頭運動

請輸出：

【A. 角色出圖提示詞】
- 主角
- 反派
- 關鍵配角

【B. 場景出圖提示詞】
- 第一集核心場景 1
- 第一集核心場景 2
- 第一集核心場景 3

【C. 視頻提示詞】
- 3 條關鍵鏡頭視頻 prompt

每條視頻 prompt 必須包含：
- 場景
- 人物
- 動作
- 情緒
- 光線
- 運鏡
- 畫面風格

資料如下：
{{貼上角色設定 + 場景設定 + 分鏡}}
```

## 最推薦的實戰方式

如果你想真的做到“給一句話就穩定生成”，我建議你固定用這個鏈路：

1. 先收集 20-50 條爆款短劇標題和簡介
2. 用「爆款樣本分析 Prompt」提煉公式
3. 用「題材升級 Prompt」把一句話升級
4. 用「系列總綱 Prompt」生成 8 集
5. 用「第一集腳本 Prompt」寫第一集
6. 用「分鏡拆解 Prompt」拆鏡頭
7. 用「出圖 / 出視頻 Prompt」做畫面

這樣會比一條 prompt 從頭幹到尾穩很多。
