# Review Focus

Use this file when you need quick criteria for second-pass script review.

## Opening 3 Seconds

- Is there an immediate image, humiliation, danger, or loss?
- Can a viewer understand the emotional problem instantly?
- Is the opening visual enough to stop scrolling?

## Episode-End Hook

- Does the final beat create a next-click urge?
- Does it open a new question, not just end loudly?
- Is it stronger than the previous 10 seconds?
- Is it a real card point, or just a loud stop?

## Pain / Payoff / Reversal

- Pain should be specific, visible, and costly
- Payoff should hit the same wound that hurt earlier
- Reversal should change understanding, not just add misery
- "almost got it, then lost it" can be a useful fake card point, but do not confuse it with a true payoff

## Distinctiveness

- Would this feel like a standard workplace comeback?
- Would a viewer predict the shape too early?
- Is there one image or setup that people would remember?
- Are the episode hooks all the same flavor?

## Front 10 Percent

- Does the story get the audience in quickly?
- Is the protagonist goal clear early?
- Is pressure applied from multiple sides?
- Is there an early card point strong enough to keep scrolling viewers?

## Shootability

- Can the scene be pictured immediately?
- Are actions visible?
- Are there too many locations?
- Is the script too dependent on explanation instead of behavior?

## AI-Tell Texture

- Are paragraphs too equal in length?
- Are hedge words repeated too much?
- Are transition words too formulaic?
- Do too many sentences begin the same way?

## Continuity Safety

- Do not recommend changes that break later episodes
- Prefer emphasis changes over plot changes
- If a true fix requires plot surgery, mark it clearly instead of sneaking it in
