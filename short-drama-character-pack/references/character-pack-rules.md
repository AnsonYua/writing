# Character Pack Rules

Use this file when extracting character packs from scripts.

## Prefer These

- repeatable clothing
- memorable silhouette
- one or two facial cues
- visible class markers
- stable emotional habits
- easy-to-draw visual anchors

## Avoid These

- flowery beauty writing
- contradictory physical details
- too many optional accessories
- vague phrases like "good-looking" without specifics
- personality claims with no support from the script

## Good Continuity Locks

- keep age presentation stable
- keep body type stable
- keep hairstyle stable unless the script changes it
- keep class/status markers stable
- keep voice energy stable

## Useful Quick Prompt Formula

Use:

- age band
- gender presentation
- body frame
- face / eyes / hair
- default clothing
- emotional aura
- one anchor object or feature

Keep it short enough for reuse.
