# Shot Value Rules

Use these rules when a scene is structurally sound but the planned panels still feel weak, generic, or low-hook.

## Core Principle

A good panel is not only clear.

A good panel is:

- instantly readable
- emotionally charged
- commercially attractive
- stronger than the other valid freeze-frame options

## High-Value Panel Types

### Proof

The viewer should understand the evidence at first glance.

Prefer:

- one screen showing several suspicious items together
- one visible arrangement that makes the pattern obvious
- proof close enough to read as the panel center

Avoid:

- generic "character checking phone" panels
- proof hidden inside vague description
- tiny proof buried in a wider emotional frame

### Reaction

The reaction should stay visibly attached to the trigger.

Prefer:

- face plus phone
- body freeze plus object contact
- eyes still locked where the shock came from

Avoid:

- generic stunned face
- reaction with no readable cause
- reaction that already breaks continuity with the next checking panel

### Chemistry

Chemistry comes from relation, not beauty alone.

Prefer:

- charged distance
- asymmetry
- near-contact
- blocked contact
- one person entering the other's emotional space

Avoid:

- flat symmetrical two-shots
- generic attractive posing
- "they are standing together" without tension

### Humiliation

Humiliation should be socially visible.

Prefer:

- crowd pressure
- center ownership by the aggressor
- exposed posture from the victim
- visible inequality in spacing or eye level

Avoid:

- shame existing only in caption or dialogue memory
- private-feeling panels when the scene value is public exposure

### Danger

Danger should look immediate and directional.

Prefer:

- threat relation
- trapped spacing
- blocked exit
- looming body or object relation

Avoid:

- generic suspense atmosphere
- threat without visible pressure

### Reversal

Reversal panels should show a changed understanding, not only a louder emotion.

Prefer:

- new proof arrangement
- a visible power flip
- a new center of control in frame

Avoid:

- more panic without new information
- escalation that looks identical to the previous beat

## Selection Rules

### Thumbnail Value

If the image is small, the panel should still read.

Keep:

- one center
- one payload
- one strong relation

Cut:

- clutter
- side business
- decorative background value that weakens the read

### Hotter Choice

When two panels are both valid, choose the hotter one:

- more tension
- more exposure
- more payoff
- more hook

### Beauty Is Not Enough

Do not keep a pretty frame if it lowers:

- proof clarity
- power readability
- romantic tension
- humiliation force
- ending hook strength
