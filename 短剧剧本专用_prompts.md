# 短剧剧本专用 Prompts

這一版只專注一件事：

- 寫「爆款短劇劇本」

不包含：

- 出圖 prompt
- 分鏡 prompt
- 視頻 prompt

另外，我已經把你提到的兩個核心能力補進去了：

- 去 AI 化
- Self-review / Auditor 自審

---

## 最推薦用法

最穩的流程不是一條 prompt 直接到底，而是：

1. 題材升級
2. 系列總綱
3. 第一集劇本
4. 去 AI 化改寫
5. 自審
6. 根據自審再修一版

如果你懶得分步，也可以直接用最下面那條「一鍵版」。

---

## 1. 題材升級 Prompt

```text
你是爆款短劇策劃師。請把我的一句話創意，升級成更適合短視頻平台的爆款短劇概念。

要求：
- 保留原始核心
- 強化身份反差
- 強化衝突
- 強化羞辱、秘密、誤會、背叛、反殺、逆襲中的至少兩項
- 更像會火的短劇，不像傳統小說
- 直接面向連載短劇，不要寫成電影

請輸出：
1. 原始創意的弱點
2. 3 個升級方向
3. 最推薦版本
4. 一句話高概念
5. 核心爽點
6. 適合的受眾

我的一句話創意：
{{一句話}}
```

---

## 2. 系列總綱 Prompt

```text
你是專做爆款短劇的編劇。請根據下面的一句話高概念，設計一個 8 集短劇系列，每集約 45-90 秒。

硬性要求：
- 第 1 集前 3 秒必須有鉤子
- 每集都必須推進新衝突
- 至少每 2 集有一次反轉
- 每集結尾都要留強懸念
- 整體要像抖音 / Reels / Shorts 爆款短劇，不像傳統電視劇

請輸出：
1. 題材定位
2. 爆款標籤
3. 主角 / 反派 / 關鍵配角
4. 三幕式總綱
5. 8 集分集設計

每集包含：
- 集標題
- 開場鉤子
- 本集衝突
- 本集反轉
- 結尾懸念

一句話高概念如下：
{{高概念}}
```

## 3. 第一集劇本 Prompt

```text
你是短劇編劇。請根據下面的設定，寫出第一集完整短劇劇本。

要求：
- 全集控制在 45-90 秒節奏感
- 場景限制在 2-4 個
- 對白短、狠、能剪輯
- 每一場都要推進衝突
- 優先用動作、對話、反應來呈現，不要用旁白長篇解釋
- 結尾必須停在最強懸念處
- 整體像爆款短劇，不像小說正文

請輸出：
1. 第一集目標
2. 場景列表
3. 完整劇本

完整劇本格式：
- 場景編號
- 地點
- 出場人物
- 畫面動作
- 對白
- 情緒變化
- 本場鉤子

故事設定如下：
{{總綱 + 第一集梗概}}
```

## 4. 去 AI 化改寫 Prompt

這一步就是把“像 AI 寫的”那種味道打掉。

```text
你是專門負責「去 AI 味」的短劇修稿編輯。請改寫下面的短劇劇本，目標是保留劇情和爽點，但讓語言、對白、節奏更像真人商業編劇寫的爆款短劇。

你必須遵守以下規則：

1. 刪掉 AI 味重的表達：
- 不要空泛抒情
- 不要總結式語言
- 不要說教感
- 不要分析報告式句子
- 不要“她感到非常震驚”“他陷入了沉思”這種直白總結

2. 用可拍的表現代替解釋：
- 多寫動作
- 多寫停頓、表情、打斷、反問
- 多寫當下衝突
- 少寫背景介紹

3. 對白去 AI 化：
- 對白更短
- 更口語
- 更像人在吵架、逼問、試探、翻臉
- 避免每個人說話都一個腔調

4. 節奏去 AI 化：
- 刪掉拖沓句
- 刪掉重複信息
- 每段都要更接近可直接拍攝的短劇節奏

5. 保持以下內容不變：
- 核心衝突不變
- 角色關係不變
- 爆點不變
- 結尾懸念不變

輸出格式：
1. 先列出原稿最重的 5 個 AI 味問題
2. 再輸出改寫後完整劇本
3. 最後簡短說明你做了哪些去 AI 化處理

待改寫劇本如下：
{{劇本}}
```

## 5. Self-Review / Auditor Prompt

這一步是模仿 repo 裡那種 Writer -> Auditor -> Reviser 的自審流程。

```text
你現在不是編劇，而是短劇審稿員 / Auditor。請嚴格審查下面這份短劇劇本，找出它不夠像爆款短劇、或仍然帶有 AI 痕跡的地方。

請從以下維度審查：

1. 鉤子是否夠狠
2. 開場 3 秒是否抓人
3. 主角是否有清晰欲望
4. 反派是否有壓迫感
5. 衝突是否推進太慢
6. 是否有多餘解釋
7. 是否有 AI 味重的台詞
8. 角色說話是否同質化
9. 反轉是否不夠炸
10. 結尾懸念是否不夠想追
11. 是否像短劇而不像小說
12. 是否有拖沓或重複

輸出格式：

【總評】
- 一句話評價這稿能不能打

【問題清單】
逐條列出問題，格式如下：
- 問題類型
- 出現位置
- 為什麼這會削弱爆款感
- 修改建議

【必改 3 項】
列出最影響成片效果的 3 個問題

【修訂指令】
把問題濃縮成一段可直接丟給編劇重寫的指令

待審劇本如下：
{{劇本}}
```

## 6. 根據自審再修稿 Prompt

```text
你是短劇主編劇。請根據下面的 Auditor 審稿意見，重寫這份劇本。

要求：
- 優先修正「必改 3 項」
- 保留原本有效的爆點
- 對白更短、更狠
- 節奏更快
- 反轉更清楚
- 結尾懸念更強
- 繼續保持去 AI 化

請輸出：
1. 修稿策略
2. 修訂後完整劇本

原劇本：
{{原劇本}}

Auditor 意見：
{{審稿結果}}
```

## 7. 一鍵版終極 Prompt

如果你懶得分 6 步，就用這條。

```text
你現在是一個「爆款短劇劇本系統」，同時具備：
- 爆款策劃能力
- 短劇編劇能力
- 去 AI 化改寫能力
- Auditor 自審能力

你的任務是：
把我給你的一句話創意，生成一份適合短視頻平台的爆款短劇劇本，並在輸出前先自行完成：
1. 題材升級
2. 系列總綱
3. 第一集劇本
4. 去 AI 化改寫
5. 自審
6. 根據自審再修一版

硬性規則：
- 只專注劇本，不要輸出分鏡、出圖、視頻 prompt
- 開頭 3 秒必須有鉤子
- 每一場都要推進衝突
- 至少包含一次強反轉
- 對白短、狠、口語化
- 多用動作和對話呈現，少用解釋
- 必須去 AI 味，避免空泛、總結、說教、報告式語氣
- 最終成品要像真人商業編劇寫的短劇，不像 AI 草稿

請按以下格式輸出：

【1. 題材升級結果】
- 一句話高概念
- 爆款標籤
- 核心爽點

【2. 8 集總綱】
每集包含：
- 集標題
- 開場鉤子
- 本集衝突
- 結尾懸念

【3. 第一集最終劇本】
格式：
- 場景編號
- 地點
- 出場人物
- 畫面動作
- 對白
- 情緒變化
- 本場鉤子

【4. 自審摘要】
- 這版原本最大的 3 個問題是什麼
- 你怎麼修掉它們
- 最終這版為什麼更像爆款短劇

我的一句話創意是：
{{一句話}}
```

## 最後提醒

如果你真的要把這套拿去穩定量產，最重要的不是把 prompt 寫得更長，而是固定兩個機制：

1. 固定你的「爆款元素庫」
2. 固定你的「Auditor 標準」

因為真正讓輸出穩的，通常不是第一次寫，而是：

- 寫完之後能不能抓出 AI 味
- 抓出來之後能不能真的改掉
